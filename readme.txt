== Installation ==

### From Gutenberg Editor:
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Advanced Image**
3. Click on the **Advanced Image** to add the block

### Download & Upload:
1. Download the **Advanced Image Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`advanced-image-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

### Manually:
1. Download and upload the **Advanced Image Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
