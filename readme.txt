== Installation ==

### From Gutenberg Editor:
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Content Slider**
3. Click on the **Content Slider** to add the block

### Download & Upload:
1. Download the **Content Slider** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`content-slider.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

### Manually:
1. Download and upload the **Content Slider** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
